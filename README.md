Restaurant Ordering System 
 
 Features 
- Interactive restaurant menu with food images 
- Real-time order processing 
- Customer order management 
- Database integration for data persistence 
 
 Tech Stack 
- **Language**: C++ 
- **Database**: SQLite 
- **Build System**: qmake 
 
 Project Structure 
- `main.cpp` - Application entry point and business logic 
- `mainwindow.cpp/h` - GUI implementation  
- `images/` - Food item images and assets 
- `restaurant.db` - SQLite database file 

Key Capabilities 
- Handles concurrent order processing 
- Efficient database operations 
- User-friendly interface design 
- Scalable architecture for restaurant operations
